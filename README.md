# Week 1 Assignment: Multi-Page Static Website

## Objective

Build and deploy a multi-page static website using only HTML and CSS.  
JavaScript NOT required.

Publish the site on Netlify and submit the deployed link as a comment in the Google Classroom module (under Week1 Day 4 - LAB).

Your task is to reorganize content from an existing informational website into your own structured multi-page website.  
You are not designing a creative site. You are restructuring content using proper HTML semantics and CSS.

You may choose any topic you like (country, university, company, travel destination, hobby, etc.).  
You do not need to recreate an entire website. Select a reasonable amount of content and reorganize it across multiple pages.
